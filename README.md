# School-management

## Min Requirements

- PHP > 8.1
- Laravel 10.x
- Composer 2.x

## How to setup

### Install Composer

```bash
git clone https://github.com/sadmanshuvo1995/eduvison.git
```

### Go to that folder

```bash
cd eduvison
```

### Env file

```bash
Create `.env` file by copying `.env.example`
```

### Run migrations with seeder

```bash
php artisan migrate:fresh --seed
```

### Install composer

```bash
composer install
```

### Serve

```bash
php artisan serve
```

### Browser

```bash
http://localhost:8000
```
